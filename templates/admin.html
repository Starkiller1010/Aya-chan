Aya-chan Admin Commands:

Description - >Command
  • Adds a specific role to user(s) {role username...} - giveRole
  • Removes a specific role from user(s) {role username...} - removeRole
  • Clears erbs linked account database of all entries, requires dev password {password} - deleteAllErbsLinkedAccounts
  • Gets an Erbs username for the user passed in if they exist in the server or have linked their account - getUserErbsAccountName
  • Gets all Erbs usernames that are linked - getAllErbsAccountNames
